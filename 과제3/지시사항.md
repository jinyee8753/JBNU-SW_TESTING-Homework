# 과제3 – CI

기존 `classify()` 메서드는 삼각형의 대표 유형 하나만 반환한다. 삼각형의 복수 속성을 함께 확인할 수 있도록 `getTypeFlags()` 기능을 추가하시오.

- (3, 3, 3) : equilateral
- (3, 3, 4) : isosceles
- (3, 4, 5) : right-angled, scalene
- (4, 5, 6) : scalene

(1) `getTypeFlags()`에 대한 unit testing 수행 추가 [Mocking 포함해야 함. 시험케이스는 가능한 경우의 삼각형과, 경계를 확인하는 시험케이스 필수]

(2) Integration testing을 위한 test case [단위시험의 시험케이스 + 통합시험을 위한 추가 시험케이스 반드시 추가]

(3) git repository의 삼각형 판별 프로그램에 통합

(4) integration testing 수행 및 통과 확인(build success)
